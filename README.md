# Júlia Eulâmpio — Portfólio de Fotografia

Site responsivo de portfólio feito com HTML, CSS e JavaScript puros.

## Como abrir

Abra `index.html` diretamente no navegador ou use um servidor local:

```sh
python -m http.server 8000
```

Depois acesse `http://localhost:8000`.

## Personalização

- Troque o nome e os textos diretamente em `index.html`.
- O conteúdo, os preços e as fotografias seguem o catálogo fornecido pela fotógrafa.
- Os links de telefone, Instagram e WhatsApp já apontam para os contatos do catálogo.
