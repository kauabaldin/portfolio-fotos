# Portfólio Júlia Eulâmpio

Site desenvolvido para apresentar o trabalho profissional de **Júlia Eulâmpio**, reunindo informações sobre seus serviços, trabalhos realizados, catálogo de preços e formas de contato em um único lugar.

## Sobre o projeto

O projeto tem como objetivo criar um portfólio moderno, visual e de fácil navegação, permitindo que possíveis clientes conheçam melhor o trabalho da profissional e encontrem rapidamente as informações necessárias para solicitar um orçamento.

O site está sendo desenvolvido com foco em:

- Apresentação profissional;
- Navegação simples e intuitiva;
- Valorização das fotografias e trabalhos realizados;
- Compatibilidade com computadores, tablets e celulares;
- Facilidade de contato com a profissional;
- Carregamento rápido e boa experiência de uso.

## Funcionalidades

Atualmente, o projeto contempla ou está sendo preparado para receber:

- Página inicial com apresentação profissional;
- Galeria de trabalhos realizados;
- Exibição dos serviços oferecidos;
- Acesso ao catálogo de preços;
- Informações sobre a profissional;
- Botões de contato e redes sociais;
- Integração com WhatsApp;
- Layout responsivo para diferentes tamanhos de tela;
- Animações e interações utilizando JavaScript;
- Melhorias de acessibilidade e otimização para mecanismos de busca.

## Tecnologias utilizadas

- HTML5;
- CSS3;
- JavaScript;
- Git e GitHub para versionamento do projeto.
